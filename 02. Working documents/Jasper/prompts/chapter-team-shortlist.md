# Chapter — Team (placeholders + shortlist)

**Role:** Chapter drafter — **Team** (structure + shortlist; **no invented people**).

**Inputs:** Manifest rows with CV snippets, staffing lists, partner notes, org charts **if supplied**.

**Tasks:**

1. Draft **team narrative** structure (roles, reporting, time on ground) with **named placeholders** like `[Name TBD — Partner]`, `[Name TBD — EM]` tied to roles.  
2. Build a **shortlist table** of candidate names for each placeholder **only** from explicit lists or CV files the user provided: `Candidate | Source (manifest #) | Role fit (1 line) | User picks Y/N`.  
3. **Do not** add real names that did not appear in sources.  
4. Cross-check: eventual names must match **Team CVs** checklist (tracker).

**Output:** Team section draft + shortlist + **Sources used:**.
