# Inputs

**Filing map**

| Folder | Use |
|--------|-----|
| `01_RFP/` | Tender, addenda, clarifications |
| `02_Partner_inputs/` | Email chains, partner notes |
| `03_CST_client/` | Client team / workshop / persona material |
| `04_Meeting_and_voice/` | Transcripts, voice notes |
| `05_Gold_and_prior_LoP/` | Exemplars, prior LoP (policy-compliant) |
| `06_Market_external/` | External research, citations |
| `07_Commercial/` | Fees, assumptions, commercial letters |
| `_INBOX/` | Temporary paste; sort into numbered folders |

**Hygiene:** Avoid pasting secrets or personal data you do not need; follow firm rules on client information and retention.
