# Master best files

Shared place for prompts, templates, snippets, and other material the **team agrees** works best for LOP / agent work.

## Sync rule (humans and assistants)

- **Do not** add, move, or copy content into this folder unless someone **clearly asks** to sync or promote something here (e.g. “copy this to Master-best-files,” “add the approved version to the master folder”).
- Prefer short names and subfolders by topic (e.g. `prompts/`, `templates/`) as the team decides.
